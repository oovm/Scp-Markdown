# SCP-2772
                        




SSTV信号传讯045的截图。已解码的文字是“这是巧合还是[未解码]见到美丽中的美丽？”



**项目编号：** SCP-2772

**项目等级：** Euclid

**特殊收容措施：** 由于项目的不可移动性，一栋带有隔音装置的收容单位已用于收容SCP-2772。基金会已经购买了SCP-2772周围的全部设施，并且由密码分析特遣小队2772-08驻扎。密码分析特遣小队2772-08同时负责收容保全措施。任何未经授权的尝试进入设施的个体将会被拘留和审问，以确认是否与SCP-2772或者2772号事件有关。

SCP-2772的图像以及音频将被记录，并送往并由密码分析特遣小队2772-08解码。所有的存档已被存放于文件2772-01至-13.任何未经安全等级4或以上的人员批准的关于SCP-2772的测试将由密码分析特遣小队2772-08负责。SCP-2772禁止被用于测试于动物或者人类上。

**描述：** SCP-2772是一个位于██████图书馆某处大厅内不可移动的空间异象。该项目大约有五米长，两米宽，七米高，占据了大厅内97%的空间。SCP-2772通常处于一种无法触摸、类似于电视花屏（television static）的视觉现象。最精确的有关穿行于SCP-2272内的感受是“一种触摸映像管电视产生轻微麻痹感。”SCP-2772内的家具地板还有墙壁会随机变换颜色，色调，以及亮度。在SCP-2772内的固态物体如家具和装饰会以非正常的方式回应任何物理性干涉。固体会变得不可触摸，自行垂直移动，或者具有弹性的同时被彻底固定在地板上。在SCP-2772内存放超过十二小时的水一样也会带有类似的性质，包括改变颜色，透明性和黏性。

每过七十二小时，SCP-2772会播放一段由窄带电视（SSTV）讯号和录音段落组成的信号。目前SSTV信号已确认是由SCP-2772自身特异性质所致，录音段落则被发现由至少由六百七十四个不同的来源剪辑而成。SSTV呈现的视频信号似乎是由内含密码的艺术加密文本而组成的图像。所有出现的SSTV信号以及录音片段都为文字组成，并由一种目前未知的加密而成。

事件2772：在2/█/20██六点十二分，一支白玫瑰和一张卡出现在SCP-2772前。由于之前SCP-2772从未有显现物质过，目前理论上同意该事件由SCP-2772之外的特异性质而产生。没有任何负责人员在此时发现任何特异事件。在卡片上的内容是由蓝色墨水，用整洁的手写体而写成。内容包括未加密过的录音事件482的所有对话，如下，以及被认为是全部其他所有SSTV传输画面的解密密码。解码过程目前仍在继续。

> 我做不到，我没法删掉数据，虽然现在和做到没两样。也许我们从来没有在一起过，但是对我来说不是。
之后没有你的日子会变得很孤单。但那些我们曾在一起的日子如果没有你只能更糟。
如果在哪里有神明，谁能听到一个不存在者的祷告，
如果在此之后仍有来生，
希望与君仍能再相会。
>
