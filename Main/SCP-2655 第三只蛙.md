# SCP-2655
                        


**项目编号：** SCP-2655

**项目等级：** Neutralized Euclid

**特殊收容措施：**  SCP-2655应当被收容在一个0.5 x 0.7 x 0.5 m的玻璃箱中，需要以小甲虫和苍蝇进行喂养。 玻璃箱左侧的水体应每两周进行一次更换。

从9/23/13开始，SCP-2655应当被收容在3 x 3 x 2 m的一个用铅板进行加强的标准收容间中。 所有进入SCP-2655所在房间的人员需要穿着含铅保护服。

**描述：** SCP-2655是一只青蛙，该种类通常被称为小雨蛙。该项目有26毫米长，主要呈棕褐色，前腿呈橄榄色和棕色斑纹。该项目生物学上类似与普通小雨蛙，但在背面以未知的墨水书写了数字三（3）以及在该项目感知当前时间为夜晚时会出现生物发光效果。 该生物发光效应已被证明有轻微放射性。在SCP-2655被初次收容时，放射性测量值为可忽略的0.03 Ci。 这个数字在初次观察后每六天增加5.1％。因为当前没有任何证据证明该项目在后天进行过任何手术或者实验，这种效应被假设自该项目出生以来一直存在。

在9/23/13，SCP-2655已被宣布无效化。 该项目在下午5:17因年老自然死亡。 辐射似乎对宿主青蛙没有影响。

SCP-2655在7/14/13被收容，一名工作人员收到消息称有一只发光的青蛙在加利福尼亚州███████████的郊区。 SCP-2655在调查其所在家庭后被收容。

该项目的所有者，一名九岁的男性，名叫Jack·███████，在接受调查时说，他一年前就发现了该项目，该项目当时就处于这个状态。

在9/23/13（见附录2655-I）的事件之后，SCP-2655的编号已被重新分配给其前任主人Jack。 他背后的数字四（4）以相同的未知墨水写成，晚上以类似于前SCP-2655的方式发光。 与其不同的是，新的SCP-2655的辐射量十分显著，从2 Ci开始，每六天增加7.3％。

> 我不认为人们意识到了这是多么危险。 9/23/13的事件让我们失去了很多人。 它在以一个指数级的增长率增长。虽然只是一个小的指数，但指数级就是指数级，总有一天我们用来收容2655的铅加固房间也会挡不住辐射的。
> 
> 我正在要求将该项目升级为Euclid。 这可能会被拒绝，因为他并不是一个威胁，但至少会引起人们的注意。 safe级别还不够。 我不想被迫射杀一个孩子只因为我们用光了资金。
> 
> Euclid级的SCP可以拿到比Safe级多至少三倍的资金。 考虑保障我们在一到两年内的生命安全的所有设备，我们没办法用这一点资金维持收容间运作。
-Dr. Kolibri
> 




从9/30/14开始，附录2655-I中的事件又一次发生了，研究员████被称为新的SCP-2655。 她与之前的SCP-2655具有相似的属性，不同之处在于她背部的数字五（5）和辐射量的增加，从7 Ci开始增加，每五天增加9％。

研究员████在9/30/14的事件中参与了六分钟时间。在此之前，她负责给SCP-2655的第二个实体投递食物。 其对于SCP-2655的第二个实体的总暴露时间估计为二十七分钟。

> 以下是关于SCP-2655最初实体死亡事件的日志。
> 


> **8:17 PM** : SCP-2655已经死了三个小时。 辐射量从0.05 Ci增加到10 Ci。
> 


> **8:20 PM** : 安保人员B1到达现场。 含铅保护服被分发给所有六名人员。
> 


> **8:21 PM** : SCP-2655的尸体从水箱中消失。安保人员B2被任命进行定位任务。 B1被指示留在水箱附近，并提供任何新的信息。
> 


> **8:27 PM** : SCP-2655被位于Site 22的自助餐厅的安保人员B2在Jack███████的裤子口袋内发现。Jack意识到了项目的出现，并在B2到来之前接触了大约十秒钟。
> 


> **8:28 PM** : Jack███████开始出现癫痫发作的症状并伴随着有着43 Ci辐射量的1.2秒伽马脉冲。
> 


> **8:29 PM** : Jack███████被注射镇静剂。伽马脉冲停止。 十五名旁观者收到了严重的放射性辐射，导致六人死亡。
> 


> **8:47 PM** : SCP-2655的Neutralized状态被撤销，并重新建立了收容。 Jack███████在可预见的未来被转移到铅加固的单位进行收容。
>
